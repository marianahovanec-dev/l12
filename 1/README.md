Модифікована програма виконує кілька ключових кроків. Спочатку користувач вводить дані про 8 товарів (назва, виробник, вартість і 5 розмірів) у масив структур `Price`. Далі, цей масив структур **сортується за спаданням вартості** товару за допомогою алгоритму сортування вибором. Після сортування, функція `saveToFile` використовує файловий потік `ofstream` для **запису відсортованого списку у файл `results.txt`** у вигляді акуратної таблиці, забезпечуючи збереження введених даних. Нарешті, програма пропонує користувачеві ввести назву та розмір для пошуку, здійснює лінійний пошук по відсортованому масиву структур і виводить повну інформацію про знайдений товар на консоль.
